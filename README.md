A versão do node.js usada é 18.16.0.


O 'banco de dados' é o arquivo __livros.json__
___

Para executar o projeto:

Entre na pasta app e use o seguinte comando para instalar os modulos
```
node install
```

Para executar o projeto
```
node app.js
```

Ou para usar o modulo nodemon
```
npx nodemon app.js 
```
